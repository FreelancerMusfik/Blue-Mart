# Blue Mart - Modern E-commerce Checkout System

Blue Mart is a fully responsive, lightweight, and user-friendly e-commerce landing page featuring a dynamic shopping cart and a streamlined checkout process. This project focuses on providing a seamless user experience from product browsing to order confirmation.

## 🚀 Features

- **Dynamic Shopping Cart:** Add or remove products with real-time price updates.
- **Advanced Checkout Modal:** A clean, two-column layout for order summary and shipping details.
- **Smart Coupon System:** Implement discount codes (e.g., `SAVE10`) with instant validation.
- **Order ID Generator:** Automatically generates unique tracking IDs (e.g., `BM-450`) upon successful purchase.
- **Search Filter:** Search by Keyword to search products.
- **Price Filter:** Price High to low & Price Low to high filtered.
- **Category Filter:** filtered by most popular categories.
- **Product Details Modal:** click any product than see here a modal popup Product Details.
- **Wishlist Functionality:** Users can mark their favorite items with a toggle heart icon.
- **Fully Responsive Design:** Optimized for Desktop, Tablet, and Mobile using CSS Flexbox and Grid.
- **Form Validation:** Ensures all required shipping information is collected before order processing.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3:** Custom styling with CSS Variables, Flexbox, and Grid.
- **JavaScript (ES6+):** Functional programming for cart logic, DOM manipulation, and event handling.
